# Rockeseat Project

This is an individual project in order to improve my learning

## ⚙️ Installation

Install Dependencies

```bash
  npm install
```

Run Project

```bash
  npm run dev
```

Generate build project

```bash
  npm Install
```

## 🖥️ Technologies Used

- React (18.2.0)
- Typescript (4.9.3)

## 📚 Libraries Used

- Date-fns
- Immer
- React Router Dom
- Styled Components
- Zod
- Phosphor React
- Eslint

## 🚀 Features

- Route creation
- Styling with external library
- Use Reducer hook
- Use API Context
- Use State hook
- Field validation
- Use local storage
